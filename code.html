Runner's-Hi implements a user-generated database to create a graph with a multiple Y-axes, quantifying the relationship between a user's mood before and after physical activity. Runner's-Hi users can post to, edit, and delete their running stats in a two-part form. The "See Stats" page displays detailed entries for completed runs, and aggregates average mood-differences from before and after to display an overall mood change average percentage.

To complete this application to specifications, we utilized the following: Vanilla JavaScript, External APIs, Github, CSS/Flexbox, Handling user events, Implementing CRUD operations, Relational data, and ERD
